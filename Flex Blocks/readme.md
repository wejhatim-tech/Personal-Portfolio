# Layout Practice Project

A simple HTML and CSS layout practice project built using Flexbox.

## Project Overview

This project is a front-end practice exercise focused on:

- HTML structure
- CSS styling
- Flexbox layout
- Margin and padding
- Nested divs positioning

The layout contains:

- Top navigation bar
- Sidebar
- Main content section
- Sub content cards
- Advertisement section

## Technologies Used

- HTML5
- CSS3
- Flexbox
